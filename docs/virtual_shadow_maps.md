# Virtual Shadow Maps / Shadow 設定

## 実装概要

`Window > 環境設定` の「シャドウ」で、従来の `ShadowMap` と `Virtual ShadowMap` を実行時に切り替えられる。

- `ShadowMap`: カメラを中心に 1～4 枚の Cascade Shadow Map を構築する。カスケード数と最遠描画距離を設定可能。
- `Virtual ShadowMap`: カメラを中心に半径が 2 倍ずつ広がる 1～4 レベルのクリップマップを構築する。各物理レイヤーは 128 x 128 px のページ（2048 px レイヤーでは 16 x 16 ページ）として管理し、各レベルでカメラ周辺の中央 2 x 2 ページだけを個別にレンダリングする。
- 受光ピクセルは、ガードバンド内かつ resident な物理ページを持つ最も細かいレベルを選び、範囲外または未割り当てページでは次の粗いレベルへフォールバックする。
- カメラ位置をシャドウテクセル単位へスナップし、移動時の shimmering を抑える。
- カメラ、ライト、シャドウキャスター、設定からキャッシュキーを生成する。変更がないフレームでは VSM 深度パスを省略し、物理ページを再利用する。
- PCF 半径、深度バイアス、法線バイアスを実行時に変更可能。
- `Shadow Mask`、`Clipmap Level`、`Virtual Page` の可視化を用意している。

物理深度は `D32_FLOAT` の `Texture2DArray` に保持する。最大 8 レイヤーを確保し、VSM の最大 4 レベルとローカルライト用レイヤーを共存させる。

## 補助シャドウ

資料で紹介されている近距離補完として、スクリーンスペースの Contact Shadow を実装した。長さとステップ数を環境設定から変更できる。シャドウマップの分解能より小さい接地影を補う用途であり、画面外の遮蔽物は扱わない。

遠距離補完として `Distance Field Shadow (AABB SDF)` も選択できる。既存のワールドAABBから最大16個の解析的ボックスSDFを構築し、ディレクショナルライト方向へレイマーチする。このプロジェクトにはメッシュSDFベイカーがないため、UEのメッシュ距離フィールドより粗い近似だが、画面外の遮蔽物と遠距離を扱える。距離とステップ数を環境設定から変更できる。

## Local Height Fog

環境設定から `Local Height Fog` と `Local Sphere Fog` を最大 16 個配置できる。

- 位置、半径、密度、色を設定可能。
- Height 形状は高さフォールオフを持つ。
- 全ボリュームを Light 定数バッファへ一括登録するため、ボリューム数に応じた追加描画パスは発生しない。
- 設定は `Save/project_environment.cfg` にシャドウ設定と一緒に保存される。

## プロジェクト設定

環境設定には、バックバッファ/シーン解像度、描画 API、VSync、固定 FPS、アンチエイリアス、HDR、トーンマッピング、露光、シャドウ、Contact Shadow、Local Fog を集約している。変更は次のフレームから反映され、正常終了時または「プロジェクト設定を保存」ボタンで保存される。
